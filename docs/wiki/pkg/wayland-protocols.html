wayland-protocols                                                   version 1.21
________________________________________________________________________________

wayland-protocols contains Wayland protocols that add functionality not
available in the Wayland core protocol. Such protocols either add completely new
functionality, or extend the functionality of some other protocol either in
Wayland core, or some other protocol in wayland-protocols.

A protocol in wayland-protocols consists of a directory containing a set of XML
files containing the protocol specification, and a README file containing
detailed state and a list of maintainers. [0]


Upstream: https://wayland.freedesktop.org/
